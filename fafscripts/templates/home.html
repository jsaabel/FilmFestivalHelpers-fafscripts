This site gives you access to various scripts that let you perform common festival-related tasks in an efficient, consistent and time-saving manner. It also acts as an alternative way of exploring and quality-checking the contents of the database. {% endblock description %} {% block content %}

Please note that, in spite of my best efforts to make this site and the scripts it lets you perform as robust and user-friendly as possible, this is my first attempt at a project of this scale and complexity. Bugs can and will occur, the visual presentation is rather basic, and I rely on your input to keep improving the user experience. Don't hesitate to let me know about any errors you might encounter, functions and improvements you would like to see or any other feedback you might have!

With that being said, the Database Analyzer script is not only intended to spot common mistakes in formatting and such, but will also help avoid database errors that can prevent the scripts from performing as expected. If you can think of a check that should be integrated, please let me know.

The database explorer pulls the most important information about films, guests and events from Notion and lets you view the database entries without any disctracting clutter. You will also find related links, scripts and other helpful information on the detail pages.

The log provides some feedback as scripts are executed, and is a good first place to check if something did not go as planned. Additionally, you will find some tips and best practices in the sidebar.

Last but not least, I intend to update this site with fixes, improvements and new features regularly. You can check the What's new? section to keep up with the latest developments, and get a sneak peek of what's to come in the sidebar.

I hope you find value in this newest addition to the FAF toolbox and enjoy exploring its features! :>
